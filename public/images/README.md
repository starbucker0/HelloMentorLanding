# Images Folder

Add your logo file here. Supported formats:
- logo.png
- logo.svg
- logo.jpg
- logo.jpeg

Make sure your logo file is named `logo.png` (or update the filename in the header component).

Recommended logo dimensions:
- Minimum: 32x32 pixels
- Recommended: 64x64 pixels or higher
- Keep it square or rectangular with good proportions
